Blast.my SMS Broadcast (Malaysia).

== Description ==

Send or broadcast SMS from your Wordpress admin panel using Blast.my service.

== Frequently Asked Questions ==

= Do I need to register with Blast.my in order to use this plugin? =

Yes, you need to have API token from Blast.my. Create your account [here](https://blast.my/register).

= What if I have some other question related to Blast.my? =

Please submit an email to nurul@ifcontech.com.
